# XWIND

Models for emission line profiles from accretion disc winds, for use in XSPEC
For a non-XSPEC version, see PyXWIND


## Requirements
A working HEASOFT installation

## Installation
1. Clone the repository
2. From within XSPEC, run the compile.xcm script

## Model Descrption

Once complied there are two (will be three) models:
1. windline - Additive emisison line profile. Currently its normalisation is calculated to that expected for Fe-Kalpha
2. windconv - Convolution mode. Applies windline broadening to an input spectrum
